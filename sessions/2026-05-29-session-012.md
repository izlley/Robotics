## 3. 결정 / 발견

### Figure 워크플로우 정착
- 모든 figure는 reproducible (script로 재생성 가능)
- SVG 우선 + PNG fallback (150 dpi)
- NanumGothic 폰트로 한글 완벽 지원
- 공통 color palette로 visual coherence

### 한글 폰트 제약 발견·우회
- NanumGothic: 한글 OK, but ✓ (U+2713), ✗ (U+2717), ☓ 등 일부 unicode 미지원
- → "OK" / "NO" 텍스트로 대체. 향후 figure에 이 규칙 적용.
- Minus 기호 `−` (U+2212)도 일부 깨짐 → ASCII `-` 또는 mathtext `$-$` 사용

### Visual ↔ Text 보완 효과
- ASCII art 다이어그램은 정렬 문제 + 한글-CJK 혼용 limit
- matplotlib SVG는 alignment 완벽 + 색상·강조 풍부
- 결과: π0의 τ 분포, attention mask 같은 추상 개념이 한눈에 명확
- 사용자 학습 효율 large gain 기대

## 4. 학습한 것

### matplotlib + SVG의 robotics learning 적합성
- 분포(τ sampling), histogram (quantile)은 matplotlib native
- Architecture diagram은 FancyBboxPatch + FancyArrowPatch로 충분
- Bar chart로 ablation 결과 시각화에 강력
- → 거의 모든 paper concept를 cover 가능

### Helper module의 가치
- `_style.py`로 color/font 일관성 → paper 간 비교 자연스러움
- save_both()로 SVG + PNG 동시 출력 → 모든 환경 호환
- 같은 패턴을 향후 모든 paper에 그대로 적용 가능

### Caption 작성 규칙
- alt text: concise (구글 검색·접근성)
- italic caption: 1-2줄로 무엇을 보여주는지 + key takeaway
- → 학습자가 figure만 봐도 핵심 메시지 캐치 가능
