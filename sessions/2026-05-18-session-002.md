## 4. 학습한 것 (요약 — 자세한 건 memory/learnings/)

1. RT-2의 핵심은 새 아키텍처가 아니라 **"action도 text token"이라는 표현 결정** — symbol tuning + output constrained decoding의 robot 응용
2. Co-fine-tuning이 단순 forgetting 방지가 아니라 **adversarial regularizer**처럼 web concept을 fine-tune 단계에서 보강 (+10pp generalization)
3. RT-2의 한계 = 후속 모델의 동기:
   - Discretization → continuous action expert (π0)
   - Closed → OpenVLA
   - Single robot → cross-embodiment (X-VLA, Octo)
   - Latency → SmolVLA
