## 2. 한 일

### A. SmolVLA.md Action Expert 상세 보강 (~250줄 추가)

`reports/papers/SmolVLA.md` 의 §3.4 (Flow Matching Action Expert) 안에 **§3.4.3 "Action Expert — Per-Block 상세 구조"** 신설:

- **(1) Input 준비**: 3종류 데이터 (action token, VLM features, τ embedding) tensor shape까지
- **(2) CA block 내부**: Q from action, K/V from VLM features, AdaLN(τ) 적용, FFN, residual
- **(3) SA block 내부**: causal self-attention, action tokens끼리만, AdaLN(τ)
- **(4) Block 전체 흐름**: pseudocode로 L_a 블록 alternating
- **(5) Output**: velocity field projection
- **(6) Tensor shape cheat sheet**: 입력부터 출력까지 모든 텐서 차원 표
- **(7) 직관적 요약 다이어그램**: ASCII diagram + 한 문장 요약

특히 강조한 부분:
- VLM features는 **layer 8 출력** (마지막 16층이 아님, § 3.3 (a) layer skipping과 연계)
- VLM features는 forward pass당 **한 번만 계산되고 모든 CA block + 10 ODE step에서 reuse**
- **AdaLN(τ)** = Esser 2024 DiT 표준 기법 (시간 conditioning)
- LLM analogy: T5 decoder의 encoder cross-attention 또는 Llama with cross-attention

기존 §3.4.3 (Action Expert width)는 §3.4.4로 renumber.
